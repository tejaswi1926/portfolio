My Projects

Personal Health and Wellness Tracker

This project is a Progressive Web App (PWA) designed to help users monitor and manage their health and wellness. It allows users to track daily activities, monitor their diet, set fitness goals, and log their mood and sleep patterns. The app provides personalized insights based on the data entered, helping users stay motivated and maintain a healthy lifestyle.

The app was built using HTML, CSS, and JavaScript, with a focus on user-friendly design and responsiveness. It also leverages modern web technologies like Service Workers and IndexedDB for offline functionality and data storage.
